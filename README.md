# Expense Tracker

A React web application for tracking personal finances, featuring real-time data aggregation and CI/CD deployment.

## Live Demo
[View Live Project at](https://expense-tracker-53.netlify.app)

## Features
* **Real-time Tracking:** Add and delete expenses instantly.
* **Time-Based Analytics:** Automatically calculates Total, Monthly, and Yearly spending.
* **Category Filtering:** Sort expenses by Food, Transport, Academic, and more.
* **Responsive Design:** Optimized for both mobile and desktop viewing.

## Tech
* **Frontend:** React.js (Vite)
* **Logic:** JavaScript (ES6+) using `.filter()` and `.reduce()`
* **Styling:** CSS (Fanta.css {https://github.com/jamezmca/fantacss} by jamezmca)
* **Deployment:** Netlify with GitHub CI/CD integration
